Programming the Unconscious: 'night, Mother (Jessie) Craft

Imaging & Unconscious
Goal

Apply Programming the Unconscious to Jessie in Marsha Norman's 'night, Mother, demonstrating how an actor finds parallels for extreme psychological circumstances.

How To

1. Analyze Jessie's dominant psychological state: Victim → Responsible Self. 2. Explore the following as parallel preparation areas via Unconscious Suggestions: - Epilepsy → use any chronic physical limitation as Available Stimulus (e.g., actress uses being overweight as parallel to being defined by a condition). - Use Fragmented Imaging to create experiences of being stigmatized by the condition. 3. Four character relationships: - Mother (only other character present): use real mother + Endowments + Fragmented Imaging. - Son (not present): use parallel younger sibling with similar issues. - Husband (not present): use former husband as choice. - Father (not present): use real father; create entire parallel life with him. 4. Write character journals: all thoughts and experiences with each character throughout life; use as blueprint for imaging. 5. Scene work: Build Fragmented Imaging montages of all events leading to the curtain. 6. Encourage dreams related to Jessie's themes; re-create meaningful ones.

Who Should Do It

Actresses working on Jessie in 'night, Mother, or any actor studying the programming process for a character whose motivation is deeply psychological and interior.

Why It Works

By the time the actress addresses the play's text, she has already built an unconscious foundation: her instrument has been repeatedly exposed to the parallel realities of Jessie's life. The character's impulses arise organically from this imprinted history.

Additional Info

Available Stimulus choice (physical condition as parallel): the actress does not need to have epilepsy—she uses whatever ongoing physical or psychological condition has defined her life experience. The emotional truth of living with a defining condition is the connection to Jessie.
