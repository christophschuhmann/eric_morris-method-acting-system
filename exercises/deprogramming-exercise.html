Deprogramming the Unconscious Exercise Craft

Imaging & Unconscious
Goal

Systematically clear all programming imagery from the unconscious after completing a role, restoring the actor's authentic reality.

How To

1. Before opening the journal: recite the mantra aloud many times: 'I worked on [character] for artistic reasons. I am not like [character]. My life is different and healthier.' 2. Open the programming journal to the first entry. Read it aloud, remembering the images created. 3. Objectify each parallel against reality: contrast every invented or exaggerated element with the actual truth of your life. Say these contrasts aloud. 4. If needed, use imaging to re-create your true reality—your real home, real mother, real relationships as they actually are—to implant accurate images. 5. Work through every entry in order. Repeat over an entire week if necessary. 6. Use the hypnagogic state to plant antidotal images; order counter-dreams. 7. Know you are done when the programming no longer has emotional pull.

Who Should Do It

All actors who used unconscious programming for a role, once the production closes.

Why It Works

The unconscious accepts repeated, sincere imagery as reality. By implanting the truth with the same deliberateness used to create parallels, the actor resets the unconscious. The body's homeostatic function supports this reset naturally.

Additional Info

The mantra functions like reprogramming a computer—downloading new information to replace old conditions. The unconscious is described as brilliant and naturally truth-seeking.
