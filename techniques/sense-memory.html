Sense Memory Both

Being and DoingNo Acting Please
The process of using the five senses to memorize and re-create sensory experiences. The actor asks detailed sensory questions about an object (real or imagined) to build or rebuild it as a living reality.
Instrumental Use

Trains the senses to be alive and responsive. Daily sense memory work sensitizes the instrument and develops the ability to create realities through imagination alone.

Craft Use

Creates the specific realities needed to stimulate the emotional obligations of a scene. The most foundational choice approach. Can re-create people, places, objects, sensations, and atmospheres sensorially. Must be done daily for at least 30 minutes.

When to Use

Daily minimum 30 minutes. Used in preparation for scenes as the primary choice approach. Can be done anywhere without special equipment.
