Programming the Unconscious (Six-Step Process) Craft

Imaging & Unconscious
A comprehensive pre-performance preparation process in which the actor systematically images character-parallel experiences from their own life, using repetition, dreams, hypnagogic states, and journals to imprint those experiences in the unconscious—so the unconscious will support the character organically in performance.
Instrumental Use

Also instrumentally valuable for deepening the actor's connection to the unconscious and developing a richer life of choices over time.

Craft Use

Step 1: Read the play (3+ times: objective → dynamics → character). Start mini-journal. Step 2: Create pre-play experiences: image what happened before the play begins and between scenes, using personal parallels (selectively emphasized). Step 3: Image all character relationships: find personal parallels for each character, build Fragmented Imaging montages of parallel relationship histories. Step 4: Address written scenes: chart character's evolution scene by scene; create imaging parallel for each scene using personal experiences. Step 5: Image yourself successfully moving through all character experiences with great involvement; repeat many times; encourage into hypnagogic state and dreams. Step 6: Keep programming journal throughout. Also use: Walter Mitty Option (fantasy parallels), Symbolic Imaging of dream symbols as performance triggers, Endowments to support programming when working with other actors.

When to Use

Only for major, multifaceted roles. Begin several weeks before rehearsal if possible. Requires psychological stability and well-developed eleventh level of consciousness.
